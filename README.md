# Web Dev Assignment 3:

## Title  
**Responsive Personal Portfolio Website**

## Description  
This project is a one-page **responsive portfolio website** made using **HTML** and **CSS**.  
It has sections like **About**, **Skills**, **Projects**, and **Contact**, and uses **Flexbox**, **CSS Grid**, **media queries**, and simple **animations/transitions**.

The layout adjusts for different screen sizes (mobile, tablet, desktop).  
Created by **Rihan Khan**
---

## Steps to View Responsive Behaviour

1. Open `index.html` in any modern browser.
2. Resize the browser window or use **Inspect → Toggle device toolbar** (in Chrome DevTools) to switch between devices.

### Breakpoints Tested

- **Mobile:** width **< 768px**  
  - Hero section stacked, skills and projects in a **single column**.
- **Tablet:** width **768px – 1023px**  
  - Hero section becomes **side-by-side**, grids show **multiple columns**.
- **Desktop:** width **≥ 1024px**  
  - More spacing, larger text/images, wider **multi-column** layout.
